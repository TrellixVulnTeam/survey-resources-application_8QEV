User Management Users & Roles

Administrators can view, invite and modify user roles for the application.

 Document Management Classify blob files, photos etc.

Define document classes and acceptable file types for uploading to support programmatic file naming as well as file sorting and filtering capabilities.

Email Management Edit Email Content for Standardized Emails.

View and edit the content of standardized emails that can be sent directly through this application.

 Lookup Table Management Update lookup table values and definitions.

Administrators can update lookup tables and definitions, that are used to populate drop-downs, list boxes, radio buttons etc.

ESRI Map Management Define layers and thematic map views.

Administrators can customize the applications map service layers to generate thematic views by toggling on/off individual layers, changing layer colors and definitions, and by switching the target feature layer. In this way, the re-usable map component can be "paired" with specific application views.
